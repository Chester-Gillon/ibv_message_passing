== Introduction ==

Notes about using a Innova IPsec with an Ethernet link-layer for perfoming the switch-test function

The test environment was:
a. A PC with a 4 core i5-2310 processor
b. AlmaLinux 8.6 with a 4.18.0-372.9.1.el8.x86_64 kernel
c. A Innova IPsec with firmware 14.22.1002 and 10G FPGA version 3505
d. No specific real-time programming techniques.
e. The device is unmanaged by NetworkManager, and at Linux boot the enp1s0 network device is DOWN with no IP address allocated.
f. A tp-link T1700G-28TQ as the injection switch with:
   - 10G port 27 connected to the Innova IPsec via a QSFP to SFP+ adapter.
   - 10/100/1000M ports ports 1 and 10 connected to the NETGEAR JGS516 TL-SF1024 10/100/1000M unmanaged switch.
   - Ports 1 to 10 being untagged members of one VLAN 1001 .. 1010 each.
   - Port 27 being tagged members of all VLANs 1001 .. 1010.
