0. Introduction
===============

This tests using the vfio-pci module, but when the IOMMU is disabled.
This is as per the configuration in "3. Test with iommu disabled and using the vfio-pci driver"
of dpdk_information_results.txt, as a result of using the "enable_unsafe_noiommu_mode=1" option
to bind to the vfio-pci driver when the IOMMU is disabled.

As a result, DPDK has to use huge pages.
